# Typora

Typora 是好用极简免费的跨平台 Markdown 编辑器，支持 markdown 的标准语法，同时 Typora 中文版还支持动态预览功能，一键预览，是一款不可多得的优质 markdown 编辑器。配合 PicGo 图片上传工具，更方便。

Typora 下载地址：https://typora.io/ (opens new window)
